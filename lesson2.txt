一：
Git工作区域
Git Repository:最终确定的文件保存到仓库
暂存区：暂存已经修改的文件最后统一提交git仓库中
工作区working directory：添加、编辑、修改文件等动作

二：
从工作区到暂存区：
git status
git add hello.php
git add test.php

从暂存区到Git Repository:
git status
git commit -m “提交描述”

最后git status查看
